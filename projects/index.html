Clock

Use this app to turn your phone into a night stand clock. With a simple interface for dimming the screen brightness and changing the font and colour. For fun, the app also includes instructions to build a stand with Lego bricks.

While not a terribly complicated app, it does demonstrate various design patterns such as Key Value Observing, View Models, Protocols, Delegates, and Singletons.

You will also find a custom view, categories, and helpful utilities in the Components folder.

Timer

Get fit using this timer app. Set the number of repetitions, the duration of each rep and the amount of rest between each rep. Set up multiple timers, select one for your workout, and tap Start to get sweating.

Written mostly in Swift. Demonstrates using a bridging header to utilize Objective-C components. Uses Cocoapods to incorporate MagicalRecord component - simplifying the effort of dealing with CoreData. Also illustrates how to derive from UIControl, play sound effects using AudioServices, customize a UITableViewCell, and set up various convenience methods via categories.
